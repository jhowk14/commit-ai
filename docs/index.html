Options

-s, --setup

Interactive configuration wizard

-e, --emoji

Use Gitmoji commit format

-p, --preview

Preview message without committing

-y, --yes

Skip confirmation prompt

-c, --config

Show current configuration

-u, --undo

Undo last commit (soft reset)
